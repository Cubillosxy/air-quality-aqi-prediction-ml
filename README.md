# air-quality-aqi-prediction-ml
mall but complete air-quality prediction pipeline: API → features → models → evaluation → plots → insights.



### Getting Started

```bash
python3 -m venv .venv
#python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate


```

- Install dependencies:
```bash
pip install -r requirements.txt
pip3 install -r requirements.txt
```


### running the pipeline
```bash
python -m src.main
python3 -m src.main
```


review the images in the `plots` directory
